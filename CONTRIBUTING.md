# Contributing to Clankbrain

Thanks for your interest. Clankbrain is a solo-maintained project 
but contributions are welcome.

## What's helpful

- Bug reports — open an issue with steps to reproduce
- Skill improvements — if a skill step fails for your stack, 
  propose a fix
- New skills — if you've built a skill that works well, share it

## How to contribute

1. Fork the repo
2. Make your change
3. Open a pull request with a clear description of what changed and why

## What to avoid

- Large refactors without discussion first
- Changes to core memory files (MEMORY.md, decisions.md templates) 
  without an issue first
- Adding dependencies — zero-dependency is a hard constraint

## Questions

Open a discussion at github.com/YehudaFrankel/clankbrain/discussions
