## How to use this file

- **High** — do this session if at all possible
- **Medium** — do this week
- **Low** — do when High and Medium are clear

Move completed tasks to Done, then remove them at End Session.
If a task is Blocked — note what's blocking it. Don't leave it invisible.

This file is read at Start Session. Keep it honest — it's useless if it doesn't reflect reality.
